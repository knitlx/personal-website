### Контекст и задача
Задача проекта:
- брать HTML-документ
- разбивать его на слайды
- получать готовый файл презентации

Важно было:
* сохранить внешний вид каждого слайда
* дать возможность проверить результат до скачивания
* использовать HTML как основной формат подготовки контента

### Как работает решение
HTML-документ размечается как набор слайдов. Каждый слайд — отдельный блок страницы.

Система:
* обрабатывает HTML
* формирует изображения будущих слайдов
* собирает их в презентацию PowerPoint

Перед финальной сборкой доступен предпросмотр.

### Предпросмотр слайдов
Перед скачиванием презентации можно пролистать будущие слайды.

Это позволяет:
* проверить структуру презентации
* увидеть визуальный результат
* внести правки в HTML до сборки файла

### Техническая реализация
* веб-интерфейс для ввода HTML
* серверная обработка документа
* отдельные маршруты для предпросмотра и сборки
* формирование презентации на сервере
* выдача готового .pptx файла

Решение не зависит от сторонних сервисов и рассчитано на повторное использование.

### Что в итоге
* HTML используется как источник для презентаций
* каждый блок становится отдельным слайдом
* внешний вид сохраняется
* результат можно проверить заранее
* PowerPoint-файл готов к использованию

Инструмент подходит для сценариев, где HTML используется как универсальный формат для генерации контента.
