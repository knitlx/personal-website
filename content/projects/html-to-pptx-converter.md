      <p>Но превратить HTML в презентацию PowerPoint — <strong>нетривиальная задача</strong>. Обычно это либо ручная сборка, либо <strong>потеря внешнего вида</strong>.</p>

      <h3>Контекст и задача</h3>
      <p>Задача проекта:</p>
      <p>брать <strong>HTML-документ</strong></p>
      <p>разбивать его на <strong>слайды</strong></p>
      <p>получать <strong>готовый файл презентации</strong></p>
      <p>Важно было:</p>
      <ul>
        <li>сохранить внешний вид каждого слайда</li>
        <li>дать возможность проверить результат до скачивания</li>
        <li>использовать HTML как основной формат подготовки контента</li>
      </ul>
      <h3>Как работает решение</h3>
      <p>HTML-документ размечается как <strong>набор слайдов</strong>. Каждый слайд — отдельный блок страницы.</p>
      <p>Система:</p>
      <ul>
        <li>обрабатывает HTML</li>
        <li>формирует <strong>изображения будущих слайдов</strong></li>
        <li>собирает их в презентацию PowerPoint</li>
      </ul>
      <p>Перед финальной сборкой доступен <strong>предпросмотр</strong>.</p>
      <h3>Предпросмотр слайдов</h3>
      <p>Перед скачиванием презентации можно пролистать будущие слайды.</p>
      <p>Это позволяет:</p>
      <ul>
        <li>проверить структуру презентации</li>
        <li>увидеть визуальный результат</li>
        <li>внести правки в HTML до сборки файла</li>
      </ul>
