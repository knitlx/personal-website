### Контекст и задача

Задача проекта — упростить работу с личными финансами без потери структуры данных.

**Важно было**:

- быстро добавлять записи
- делать это в привычной среде
- не зависеть от строгого формата
- иметь доступ к сводной информации

Отдельное внимание уделялось безопасности и корректной работе с пользовательскими данными.

### Как устроено решение

Проект состоит из двух частей:

- Telegram-бот для добавления записей
- интерфейс для просмотра и анализа

Через бот данные сохраняются, через интерфейс — просматриваются и сравниваются.

### Ввод через Telegram

Telegram используется как основной способ ввода. Данные можно добавлять:

- текстом
- голосом
- фотографиями (например, чеками)

AI разбирает сообщение, определяет сумму, категорию, дату и тип операции и сохраняет данные в структурированном виде.

### Обработка и логика

Вся обработка выполняется автоматически.

Система:

- разбирает входящие сообщения
- учитывает пользовательские категории
- корректно работает с датами и временем
- сохраняет и обновляет записи

AI используется для разбора ввода, а не для советов или принятия решений за пользователя.

### Анализ и интерфейс

Для просмотра данных используется веб-интерфейс в формате Telegram Mini App.

В нём доступны:

- доходы и расходы за период
- текущий баланс
- распределение расходов по категориям
- динамика по месяцам

Есть фильтры:

- по времени
- по валюте
- по типу операций

Интерфейс рассчитан на мобильное использование и быстрый просмотр данных.

### Безопасность и доступ

Доступ осуществляется через Telegram. Каждый запрос привязан к конкретному пользователю. Данные изолированы и недоступны другим. Отдельная регистрация не требуется.

### Техническая часть

- ввод через Telegram
- автоматизированная обработка
- AI для разбора сообщений
- хранение данных в базе
- интерфейс в формате Mini App

Решение рассчитано на постоянное использование и не требует ручной обработки.

### Что в итоге

- записи добавляются быстро
- меньше лишних действий
- данные сразу доступны для просмотра
- всё находится в одном месте — в Telegram

AI используется как инструмент упрощения работы с данными, а не как самостоятельный финансовый помощник.
